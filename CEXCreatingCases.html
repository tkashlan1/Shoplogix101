Creating Cases

1. First, click on "Support Cases" button found in the navigation bar.

2. Click on the "New Case" button.

3. In the "subject" line, describe the case you are creating. For example, if you need a machine licensed, your subject can be "Licensing on saas7."

4. Adjust the "Priority" section to reflect the level of importance of the case.

5. Adjust the "Type" section to depending on the case. For instance, a licensing case should be set to "Licensing" .

6. The "Owner" of a case would generally be "Support Que" or "Dev Que"; if you would like to send the case to a specific person, type their name then
click on it when it appears.

7. You need to specify which customer the case belongs to. For example, a customer could be "Bway Metals Homerville".

8. The contact should be the go-to person if the owner of the case has any questions regarding the case.

9. Adjust the "Priority" section to reflect the level of importance of the case.

10. You should also specify the name of the machine that needs to be investigated or needs adjustments to be made in the "Machine" section.

11. In the "Description" box, you should add all the details that a owner of a case should know in order to work on the case. For example, if a machine
needs downtime reasons added, you would specify which reasons need to be added.

12. Once all the necessary information has been filled out, click on the "Email Notification" check box if this desired then "Save."
